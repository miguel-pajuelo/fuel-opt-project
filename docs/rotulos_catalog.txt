# Grouping decisions
- Repsol: REPSOL + CAMPSA + PETRONOR. CAMPSA se agrupa como red histórica Repsol; PETRONOR es filial/red Repsol.
- Cepsa: CEPSA + MOEVE + DISA. MOEVE aparece como rótulo principal actual; DISA se agrupa con Cepsa para cobertura Canarias/grupo Cepsa cuando aparezca como rótulo independiente.
- BP: BP + ARAL. ARAL se mantiene agrupado bajo BP si aparece en MINETUR.
- Shell: SHELL.
- Galp: GALP.
- TotalEnergies: TOTAL + TOTALENERGIES + ESSO. ESSO queda como alias condicionado a aparición real.
- Low-cost: PLENERGY/PLENOIL, PLENITUDE, MEROIL, PETROPRIX, AVIA, BONAREA y Q8 se mantienen como marcas independientes salvo evidencia contraria.
- Retail: ALCAMPO, CARREFOUR, EROSKI, EL CORTE INGLES/ECI y LECLERC/E.LECLERC se mantienen como marcas independientes.
- UNKNOWN: no se usa para adaptar marcas; solo indica fuente degradada o sin rótulo.
